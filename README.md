## Instalacion
1. Clonar el repositorio.
2. Crear una base de datos en MySQL con el nombre de `hotel_alura`.
3. Ejecutar el script `hotel_alura.sql` en la base de datos creada.
4. Ejecutar el archivo `hotelAlura.jar`.
